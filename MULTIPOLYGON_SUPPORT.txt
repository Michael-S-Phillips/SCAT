SCAT - MultiPolygon Support Update
===================================

Date: November 3, 2025
File Modified: scat.py (lines 2855-2922)
Status: ✓ IMPLEMENTED

CHANGE SUMMARY
==============

The SCAT polygon loading function has been updated to handle both simple Polygon
and MultiPolygon geometries when loading GeoPackage files.

ISSUE FIXED
===========

Previous Error:
  AttributeError: 'MultiPolygon' object has no attribute 'exterior'

This error occurred when loading consolidated GeoPackage files that contain
MultiPolygon geometries (merged from multiple individual polygons).

IMPLEMENTATION DETAILS
======================

Modified: load_polygons() function (scat.py)

Key Changes:
  1. Added geometry type detection: geom.geom_type == 'MultiPolygon'
  2. For MultiPolygon geometries:
     - Iterate through each component polygon (geom.geoms)
     - Process each polygon individually
     - Extract exterior coordinates for each component
     - Add each component to the polygon lists with same attributes
  3. For simple Polygon geometries:
     - Use original logic (.exterior attribute)
     - Extract and convert coordinates as before

CODE STRUCTURE
==============

The polygon loading loop now has two branches:

if geom.geom_type == 'MultiPolygon':
    # For each component polygon in MultiPolygon
    for poly in geom.geoms:
        # Extract coordinates, convert to pixel coords
        # Add to self.all_polygons, colors, spectra, etc.
        # Use same row index (i) for all metadata lookups
else:
    # Original Polygon handling logic
    # Extract coordinates, convert to pixel coords
    # Add to all lists

BEHAVIOR
========

MultiPolygon Handling:
  - Each component polygon is loaded separately
  - All components share the same attributes (color, mineral IDs, spectrum)
  - Threshold value stored in "Mineral ID 1" is preserved for all components
  - Rendering shows all component polygons in same color

Example with Consolidated File:
  - T1250_BD1300_stack_scat_consolidated.gpkg loads 6 features
  - Feature 0 = Threshold 0 = 1 MultiPolygon (863 component polygons)
  - SCAT displays all 863 components in red color
  - Feature 1 = Threshold 1 = 1 MultiPolygon (1,145 component polygons)
  - SCAT displays all 1,145 components in blue color
  - And so on for each threshold...
